# Research Note

## Problem

Deterministic event replay and first-divergence detection for trading-system debugging.

## Method

The package exposes a compact deterministic model that can be tested without external data.
This makes it useful as a building block for later simulation, benchmark, or dashboard work.

## Limits

The repository is intentionally public-safe. It avoids proprietary market data, auth tokens,
live execution hooks, and unsupported profitability claims.
